# Can AI Explain Human Emotions in Clinical Contexts? 🧠🔬

## Project Overview
This project investigates the interpretability of Deep Learning models applied to **Facial Expression Recognition (FER)**. 
We explore how different eXplainable AI (XAI) techniques can validate the use of AI as a non-invasive biomarker for neurological disorders.

## Key Features
- **Models:** CNN (ResNet) and Vision Transformers (ViT).
- **Dataset:** AffectNet.
- **XAI Techniques:** Grad-CAM, Saliency Maps, Integrated Gradients, Attention Rollout.
